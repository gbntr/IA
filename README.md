Este espaço foi criado para centralizar o versionamento dos códigos desenvolvidos ao longo do semestre. O conteúdo inclui principalmente exercícios propostos em aula.

## 🛠️ Tecnologias e Ferramentas

* **Ambiente de Desenvolvimento:** Google Colab / Jupyter Notebook
